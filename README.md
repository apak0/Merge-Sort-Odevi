# Merge-Sort-Odevi



Merge Sort Odevi
[16,21,11,8,12,22] -> Merge Sort
## Sorular

1)Yukarıdaki dizinin sort türüne göre aşamalarını yazınız.
2)Big-O gösterimini yazınız.

## Cevaplar
`1-
 [16,21,11,8,12,22]
[16,21]-[11]| [8]-[12,22]
[16]-[21]-[11] | [8]-[12]-[22]     
[16,21]-[11] | [8,12]-[22]
[11,16,21] | [8,12,22]
[8,11,12,16,21,22] Şeklinde sonuçlanır.`

 `2- Big-O gösterimi : O(nlogn)'dir.`
